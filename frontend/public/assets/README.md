# Assets Directory

## Required Files

### 1. Company Logo
**File**: `cslogo.jpeg`
- Place your CS Smart Finserve logo here
- Recommended size: 200x200 pixels minimum
- Format: JPEG or PNG
- Background: Transparent (PNG) or white

### 2. Favicon
**File**: `../favicon.ico`
- Crop the hexagon CS icon from your logo
- Create favicon with multiple sizes (16x16, 32x32, 48x48)
- Place in the public folder (one level up)

### 3. PWA Icons (Optional)
**Files**: `../logo192.png` and `../logo512.png`
- For Progressive Web App support
- Place in the public folder
